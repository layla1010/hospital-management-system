# Hospital Management System

## Overview
Hospital Management System is a Java desktop application designed to manage core hospital operations through a graphical user interface.  
The system supports managing patients, staff members, departments, medical problems, medications, treatments, and visits.

This project demonstrates object-oriented programming, modular design, exception handling, GUI development, and file-based data handling in Java.

## Key Features
- User login system
- Patient record management
- Staff member management
- Department management
- Medical problem tracking
- Treatment and medication management
- Visit record management
- CSV export functionality
- Custom exception handling
- GUI-based navigation using Java Swing

## Technologies Used
- Java
- Java Swing
- Object-Oriented Programming
- Custom Exceptions
- File Serialization
- CSV Handling
- External Java libraries for UI components

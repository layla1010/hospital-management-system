# Hospital Management System

A Java-based desktop application for managing hospital operations, including patients, staff, departments, treatments, medications, and visits.

## Technologies
- Java
- Java Swing
- Object-Oriented Programming
- Exception Handling
- CSV Handling
- File Serialization

## Features
- Login system
- Patient management
- Staff management
- Department management
- Treatment and medication management
- Visit records
- CSV export

## Project Structure
- `src/model` – system entities
- `src/control` – main system logic
- `src/view` – graphical user interface
- `src/exceptions` – custom exceptions
- `src/utils` – helper classes

## How to Run
1. Open the project in Eclipse.
2. Make sure the JAR files are added to the build path.
3. Run `LoginPage.java` from `src/view`.
